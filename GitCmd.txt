1. Recupere le depot remote en local
> git clone https://github.com/StefBibi/TestGit.git
> cd 'directory'

2. Ajout du fichier des commandes, packaging des changements, envoi vers depot
> git add .
> git commit -m "ajout d'un fichier .txt
> git push


2. Ajout d'un 2nd fichier, packaging des changements, envoi vers depot
> git add -A
> git commit -m "ajout d'un fichier .txt
> git push
